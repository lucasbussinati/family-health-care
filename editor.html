✏️ Editor da Anamnese

Como funciona: tudo acontece no seu navegador, nada é enviado para a internet aqui. Escolha a família, edite os dados, defina a senha e baixe o arquivo dados-<família>.json já criptografado. Depois é só subir esse arquivo no GitHub.
🔑 Para trocar a senha no futuro: carregue os dados da família com a senha antiga, escolha uma senha nova na seção "Gerar arquivo" e baixe o arquivo novamente.
💾 Edição aos poucos: tudo que você preencher é salvo automaticamente como rascunho neste navegador e volta sozinho quando você reabrir o editor — assim dá pra completar os dados com o tempo. Quando terminar, gere o arquivo e suba no GitHub.
